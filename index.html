Joe's Zaizi take-home challenge

I've written code to play lots of simulations of the Monty Hall problem.

The outcomes of these simulations are shown on the graph below.

The red points show the cumulative number of wins when using the "stick" strategy.

The yellow points show the cumulative number of wins when using the "switch" strategy.

The "guesses" that the simulated player makes are randomised* each time this page is loaded, so if you want to convince yourself further, try reloading the page a few times and see how the graph changes. The shape of the plots varies, but the switch strategy clearly leads to a greater number of wins.

*(pseudo-randomised using JavaScript's Math.random, which has an "approximately uniform distribution")
